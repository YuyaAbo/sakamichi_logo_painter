# sakamichi_logo_painter

![image](https://github.com/YuyaAbo/sakamichi_logo_painter/sample_image.png)

## Usage

```dart
Container(
  child: CustomPaint(
    size: Size(60, 60),
    painter: SakamichiLogoPainter(
      foregroundColor: Colors.blue,
      backgroundColor: Colors.purple,
    ),
  ),
),
```
